## Usage

Install [Docker](https://nbviewer.jupyter.org/github/as641651/GCP/blob/master/Docker/01_Docker_Installation.ipynb)

Allow execution of all shell files
```bash
chmod +x *.sh
```

### Build Image
```bash
./build.sh
```

### Run Container
```bash
./run [PATH_TO_VOLUME]
``` 

### Run jupyter notebook
```bash
jupyter-notebook --allow-root
```
